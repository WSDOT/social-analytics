Social Analytics

An analytics dashboard for twitter.

View on GitHub Download .zip Download .tar.gz

Features

  • Tweet and Mention counts
  • Follower statistics
  • Tweet search and export
  • Sentiment analytics
  • Show popular/unpopular tweets
  • Tweet Source data

Setup

  1. Install maven.
  2. Run mvn install in the project directory.
  3. Follow this guide on deploying a GWT app to a web server
  4. Set up server files found in server/

NOTE: Make sure to change HOST_URL in Consts.java, as well as adding your twitter accounts in the source (AnalyticsViewImpl.java, AnalyticsViewImpl.ui.xml and AdvSearchView.ui.xml).
